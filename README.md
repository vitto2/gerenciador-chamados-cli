# gerenciador-chamados-cli
Ferramenta de linha de comando para registrar e gerenciar chamados de suporte técnico (help desk de primeiro nível). Roda no terminal, sem interface gráfica e sem banco de dados. Simula o fluxo real de uma equipe de suporte: abrir chamado, acompanhar, atualizar status, encerrar e gerar um resumo.
